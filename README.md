# 3-Tier Docker Application

## 🚀 Overview
This project demonstrates a 3-tier architecture using Docker:

- Frontend (Nginx)
- Backend (Node.js)
- Database (MongoDB)

## 🧱 Architecture
User → Frontend → Backend → Database

## 🔧 Technologies
- Docker
- Docker Compose
- Node.js
- MongoDB
- Nginx

## ▶️ Run the Project
```bash
docker compose up -d --build
